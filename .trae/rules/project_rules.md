# 高考倒计时项目规则

**技术栈**: HTML + CSS + JS，无外部库  
**兼容性**: Chrome 61+

## 核心规则

1. **禁止引入任何外部库**（jQuery、Bootstrap、框架等），所有功能使用原生JS/CSS实现

2. **兼容性要求**：
   - 使用 ES6+ 语法（Chrome 61+ 完全支持）
   - 避免使用过新的 API，如需使用需检测兼容性
   - CSS 使用标准属性

3. **代码风格**：
   - CSS 优先使用 Flexbox 布局，Grid 作为增强
   - JS 采用 IIFE 避免全局污染
   - HTML 结构语义化

4. **文件组织**：
   - `index.html` 为入口文件
   - CSS/JS 按功能拆分，保持单一职责
   - 资源文件放 `assets/` 目录

5. **性能优化**：
   - 减少 DOM 操作，使用事件委托
   - CSS 放在头部，JS 放在 body 底部
   - 图片使用现代格式（SVG 优先）
